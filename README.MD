Voxor the Sculptor
==================

A simple Voxel sculpting tool for HTC VIVE

Contains Unity 4.3.0f3

Use
---
Left controller: color
Right controller: cut

touchpad up/down: increase/decrease size of tool
touchpad left/right: change hue (let only)
